Repositório utilizado para **testar e executar comandos do Git** no terminal (**Git Bash**) em conjunto com um **banco de dados PostgreSQL**, utilizando o **pgAdmin** como ferramenta de gerenciamento.

---

## 🎯 Objetivo

Praticar o uso do **Git** em um contexto de manipulação de arquivos relacionados a **Banco de Dados**, simulando versionamento de scripts SQL e documentos de modelagem.

---

## 📂 Conteúdo do repositório

- **Scripts SQL** utilizados para testes no PostgreSQL  
- **Arquivos de apoio** relacionados à modelagem do banco de dados  
- Histórico de commits focado na prática de comandos Git  

---
## 🛠 Tecnologias utilizadas

- Git
- Git Bash
- PostgreSQL (SQL)
- pgAdmin
